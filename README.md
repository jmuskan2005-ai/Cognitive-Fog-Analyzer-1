COGNITIVE FOG ANALYZER

# About:
A small terminal-based tool that helps you understand how “foggy” or clear your mind feels 
each day. You just enter a few basic details, and it gives you a simple fog score while saving 
everything so you can look back at your patterns anytime.

# Features:
- Easy terminal inputs 
- Calculates a personalized fog score 
- Automatically saves data to cognitive_fog_log.csv
- Gives short, human-friendly messages 
- Offers a summary of your best, worst, and average days

# How to run:
1. Install Python 3.13.7
1. Clone the repo:
   https://github.com/jmuskan2005-ai/Cognitive-Fog-Analyzer-1.git
3. Move into the project folder 
cd cognitive-fog-analyzer-1 
4. Run the program 
python Cognitive fog analyzer.py
5. Follow the questions in terminal and enter your answers: 
   - Choose if you want to add a new entry/view summary/exist
   - if choosen new entry then enter yours slept hours and rate your focus, stress level all out of 10 and enter your screen time. This will calculate your cognitive fog score out of 10 with a message for you
   - if choosen summary then that will give you the highest, lowest, and average of all till now
   - if choosen exist then you can exist it.
# Requirements:
- Python 3.7 or higher 
- Works on Windows, macOS, or Linux 
- Built-in modules only: csv, datetime, os, statistics
